# Realtime Node.js App (Express + Socket.io)

This project is a simple realtime chat application built with **Node.js**, **Express**, and **Socket.io**.
It uses **Nodemon** during development to automatically restart the server when files change.

> This project is based on lessons from **[Creative Coder YouTube Channel](https://www.youtube.com/@CreativeCoderMyanmar)**. Thanks to them for the tutorial content!
